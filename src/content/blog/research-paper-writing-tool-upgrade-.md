PaperSpine V2 是我们基于对大量用户反馈和自身实践的深入理解后推出的系统性升级版本。自第一版发布以来，PaperSpine 已获得超过 20 万次浏览、1 万多次点赞与收藏，以及数百个 GitHub star。用户反馈表明，虽然我们提供的工具在一定程度上能够满足论文写作的需求，但在面对论文、技术报告、课程作业等具体场景时，AI 还是存在着“不知道该如何写”的问题。

因此，我们决定在 V2 中进行一次全面的升级。新版不仅支持 Codex 和 Claude Code 等多种 AI 环境，还能适应期刊、会议、技术报告、综述、竞赛论文等多种写作场景。除此之外，用户现在可以利用已有初稿进行改写，也可以从零开始，通过素材文件夹逐步构建论文。这意味着 PaperSpine 不仅仅是一个“润色器”，而是一个完整的写作支持系统。

一个特别重要的升级是“写作思路矩阵”（writing_rationale_matrix.md）。这项功能代表了 V2 最大的突破。它通过显性记录“为什么这样写”的过程，帮助我们理解 AI 是如何形成论文结构与表达策略的。这样的机制不仅增强了 AI 写作的透明度，还为我们提供了学习和模仿优秀论文的路径。

在实际应用中，用户可以通过这个矩阵追踪论文的整体框架，乃至每个段落、图表和结果解释背后的逻辑。这种结构化的方式既能防止 AI 仅停留在表面润色，还能促进用户对论文写作规范的深层次理解。

PaperSpine V2 的发布，标志着我们在 AI 辅助写作工具上迈出了坚实的一步。它不仅是对现有功能的拓展，更是对写作流程的深度重构，为研究者和学习者提供了更为智能、可控的写作体验。

<!--EN-->  
PaperSpine V2 represents a significant upgrade in the realm of research paper writing tools. Following the overwhelming positive response to the first version, which garnered over 200,000 views and thousands of likes and shares, we've made substantial improvements to enhance the user experience. The latest iteration not only supports multiple AI environments such as Codex and Claude Code but also caters to various academic and technical writing scenarios.

A notable enhancement in PaperSpine V2 is the introduction of the "writing rationale matrix," a feature that provides a structured way to understand the reasoning behind the AI's writing process. This matrix allows users to trace the logic behind the structure and content of a paper, from the overall framework down to individual paragraphs, figures, and results. This innovative approach enhances transparency in AI-generated content and provides an educational tool for learning how to write effectively.

With PaperSpine V2, we aim to offer a more intelligent and controllable writing experience, empowering users to produce high-quality academic content with greater ease and precision.
