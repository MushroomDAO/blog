## 一、为什么"第一站"是一个战略命题

全球数字游民选择目的地，遵循一套清晰的逻辑：

1. **签证可行性**：能不能合法长期停留？
2. **基础设施**：网速够不够快？落地能正常工作吗？
3. **社区密度**：有没有同类？能不能获得信息和社交？
4. **生活成本**：在这里生活，性价比如何？
5. **可探索性**：这个城市/地区有没有让人想留下来的理由？

中国过去在第1点（签证）严重失分，导致几乎没有出现在全球游民的选项清单里。

但这个局面正在改变。

### 中国签证政策的历史性转折

| 政策类型 | 现状 | 对游民的意义 |
|---------|------|------------|
| 单方面免签 | 已覆盖 **47个国家**（含美、英、德、法、日、韩、澳大利亚等） | 大量游民无需提前申请签证 |
| 泰中互免签证 | 2024年3月生效，双向免签30天 | 清迈游民往返资阳**零门槛** |
| 240小时过境免签 | 2024年12月17日起升至240小时（10天），开放口岸60个 | 游民可"过境"方式在中国停留10天无需签证 |

### ⭐ 关键发现：资阳明确列入240小时过境免签范围

**2024年12月17日起，四川省过境免签停留范围扩展至11个城市：**

> **成都、乐山、德阳、遂宁、眉山、雅安、资阳、内江、自贡、泸州、宜宾**

一名美国、英国、德国、法国游民，从天府国际机场（列入过境口岸）入境后，可在**资阳合法停留10天，无需任何签证**。

这是OPC建立"全球游民第一站"最直接的政策支撑。

## 二、资阳的地理优势：天府机场枢纽

### 天府国际机场：资阳的战略资产

- **位置**：位于成都简阳，紧邻资阳市域，驾车约20-30分钟
- **规模**：规划年旅客吞吐量9,000万人次，远期1.2亿人次（全国第二大机场）
- **国际航线**：覆盖欧洲、东南亚、中东、北美主要城市

### 对国际游民的吸引力公式

> 成都的国际认知度 + 天府机场的直达便利 + 资阳更低的生活成本 + OPC的专属社区 = 极具竞争力的选项

## 三、如何实现"全球游民第一站"目标

### 3.1 基础设施层

| 需求 | 目标 |
|------|------|
| 签证信息 | OPC官网提供多语言签证指南 |
| 住宿 | 整合或建设游民公寓/联合居住 |
| 网络 | OPC保障区域500Mbps+接入 |
| 支付 | 帮助游民绑定境外卡开通支付宝/微信 |
| 语言 | OPC配备英语接待，关键信息英文化 |

### 3.2 社区层

**"30天游民体验计划"**：
- 首次入驻30天，低于市价30%优惠月租
- 配套接机服务、城市定向介绍、本地人对接
- 每周1次英文社区活动

**游民大使机制**：
- 筛选有影响力的国际游民（粉丝1万+），邀请免费驻留2周
- 要求发布内容介绍资阳，参与OPC合作活动

### 3.3 价值层：让游民从消费者变贡献者

```
传统游民目的地：游民来 → 消费 → 离开

资阳OPC模式：游民来 → 获得订单/项目 → 贡献产出 → 获得收益 → 愿意回来/推荐
```

6+8产业课题转化为游民可参与的真实需求，游民组队竞标获得真实报酬。

## 五、竞争格局与真实挑战

### 中国城市竞争分析

| 竞争者 | 优势 | 弱点 | 对资阳威胁 |
|--------|------|------|----------|
| **成都** | 国际知名度高、美食/文化吸引力强 | 生活成本高、无专属游民社区定位 | ⚠️ 中等（走高端，资阳可差异化） |
| **海南自贸港** | 政策红利、免税 | 夏季极热、政策落地慢 | ⚠️ 中等（面向金融/互联网） |
| **大理** | 已有游民文化基础、自然环境好 | 国际化程度低、无国际直飞 | 🟡 低（偏国内游民） |

**资阳的窗口期**：目前没有任何中国城市系统性建立"国际数字游民"品牌，这个窗口期估计在**2-3年内**。

### 真实障碍与解决方案

| 障碍 | 游民顾虑 | 资阳OPC应对 |
|------|---------|------------|
| **网络访问** | Google/Slack/GitHub等工具受限 | 提供合规商业网络解决方案 |
| **支付工具** | 国际信用卡不可用 | 提供入驻辅助包，帮助开通支付 |
| **语言壁垒** | 几乎无英语服务界面 | 配置英语接待，提供双语陪同 |
| **信息不透明** | 国际平台对资阳几乎无信息 | 主动在Nomad List提交资料，在Reddit发布英文内容 |
| **签证续签** | 10天过境免签之后如何合法续留？ | 提供签证咨询，明确长期停留路径 |
