apfel: Apple Silicon Mac Local AI Tool

apfel is an open-source CLI tool that lets developers run Apple Intelligence locally on Apple Silicon Macs—zero cost, zero config, fully local.

**Key Features:**
- Zero cost: No API subscription needed
- Zero config: Works out of the box
- Fully local: Privacy-preserving inference
- UNIX integration: Pipe support
- OpenAI-compatible API server
- MCP (Model Context Protocol) support
- Context management (4096 tokens)

**Requirements:**
- Apple Silicon Mac (M1/M2/M3/M4)
- macOS 26 (Tahoe) or later
- Apple Intelligence enabled

**Use Cases:**
- Shell command generation
- Code review and explanation
- Local log analysis
- Privacy-sensitive AI tasks

**Quick Start:**
```bash
brew install apfel
apfel "explain recursion"
apfel --chat
apfel --serve --port 8000
```
